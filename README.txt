To install this plugin, you have to :
1. put it in your %SF_PLUGIN_DIR% directory
2. add it into the config/extra-modules.php file
3. build the model, filters and forms, and create the database
4. activate the modules in your targetted sf1 app as proposed in the config/extra-modules.php.template provided by the plugin
5. discover new submenus in the "ticketing" menu and various modules
